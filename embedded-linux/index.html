记录嵌入式 Linux 的启动流程、驱动调试、系统构建、交叉编译和板级实践。

这里整理嵌入式 Linux 相关内容，关注从芯片上电到应用运行之间的完整链路：Bootloader、Kernel、设备树、驱动、根文件系统、交叉编译、系统裁剪和现场调试。

目标不是堆命令，而是把问题背景、定位思路、关键证据和复盘结论写清楚。
